Как запустить сайт 
1. Скачайте все архивы

2. Распакуйте скачанный ZIP-архив

3. Создайте рабочую папку
На Рабочем столе создайте новую папку и назовите её как нибудь.
4. Переместите все файлы в эту папку
Возьмите все распакованные файлы и папки, которые вышли после распаковки, и перенесите их в созданную папку   
Внутри должно оказаться:
- MarKen.html — главный файл сайта
- assets.zip (его тоже нужно распаковать!)
- audio_1.zip, audio_2.zip, audio_3.zip
- episode.zip
- gallery.zip
- trailer.zip

5. Распакуйте все ZIP-архивы внутри папки
Распакуйте каждый ZIP-файл. что должно получится:
- MarKen.htm
- папка assets
- папка audio_1
- папка audio_2
- папка audio_3
- папка episode
- папка gallery
- папка trailer

Примечание: файлы из архивов должны лежать внутри своих папок, а не просто россыпью

6. Запустите сайт
Дважды щёлкните по файлу MarKen.html — он откроется в вашем браузере.

Примечание: если сайт показывает пустые картинки или музыку не играет — значит, какой-то архив забыли распаковать или положили не в ту папку
